# Minesweeper.kt
This is a solution to the [Numeric Matrix Processor project](https://hyperskill.org/projects/87) on JetBrains Academy at hyperskill.org

### How to run
Windows:
```sh
> .\gradlew.bat --console=plain --quiet run
```
Linux:
```sh
$ ./gradlew --console=plain --quiet run
```
